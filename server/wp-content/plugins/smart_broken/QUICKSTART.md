# ⚡ Quick Start Guide

## Get Started in 5 Minutes!

### Step 1: Install (1 minute)
```bash
# Upload to WordPress plugins directory
cp -r smart-broken-link-fixer /path/to/wordpress/wp-content/plugins/

# Or via WordPress Admin:
# Plugins > Add New > Upload Plugin > Choose ZIP file
```

### Step 2: Activate (10 seconds)
- Go to WordPress Admin > Plugins
- Find "Smart Broken Link Fixer"
- Click "Activate"

### Step 3: Configure (2 minutes)
1. Go to **Broken Links > Settings**
2. Check these options:
   - ✅ Enable Automatic Scanning
   - ✅ Enable AI Suggestions
   - ✅ Scan External Links (optional)
3. Enter your email for notifications
4. Click **Save Settings**

### Step 4: First Scan (1 minute)
1. Go to **Broken Links > Dashboard**
2. Click **Start New Scan**
3. Wait for completion

### Step 5: Fix Links (1 minute)
1. Go to **Broken Links > Broken Links**
2. For each broken link:
   - Click **AI Fix** (get suggestion)
   - Click **Redirect** (apply fix)
   - Or click **Ignore** (skip)

## 🎉 Done!

Your WordPress site is now protected from broken links!

## Keyboard Shortcuts

- `Ctrl/Cmd + Click` on link = Open in new tab
- `Enter` in modal = Apply redirect
- `Esc` = Close modal
