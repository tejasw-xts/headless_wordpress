== Description ==

Smart Broken Link Fixer is an advanced WordPress plugin that goes beyond traditional broken link checkers. It not only detects broken links but also provides AI-powered suggestions and automatic redirect capabilities to fix them.

= Key Features =

* **Automatic Link Detection** - Scans your entire website for broken links
* **AI-Powered Suggestions** - Intelligently suggests replacement pages based on content similarity
* **Auto Redirect** - Automatically redirect broken links to suggested pages
* **Email Notifications** - Get notified when broken links are detected
* **Scheduled Scans** - Set up daily or weekly automatic scans
* **Detailed Reports** - View comprehensive scan history and statistics
* **Easy Management** - Simple interface to fix, ignore, or redirect broken links
* **External Link Scanning** - Option to scan external links as well
* **REST API** - Full REST API support for developers

= How It Works =

1. The plugin scans your website content for links
2. Checks each link to determine if it's broken (404, timeout, etc.)
3. Uses AI algorithms to suggest relevant replacement pages
4. Allows you to apply redirects with one click
5. Sends email notifications to keep you informed

= Perfect For =

* Website owners who want to maintain link quality
* SEO professionals managing multiple sites
* Content managers dealing with legacy content
* Anyone who wants to improve user experience

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/smart-broken-link-fixer/` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Broken Links > Settings to configure the plugin
4. Click "Start New Scan" to begin detecting broken links

== Frequently Asked Questions ==

= Does this plugin fix links automatically? =

The plugin can suggest fixes and apply redirects. You have full control over which links to fix, ignore, or redirect.

= Will it slow down my website? =

No. Scans run in the background and don't affect your website's front-end performance.

= Can I scan external links? =

Yes, you can enable external link scanning in the settings.

= How often should I scan for broken links? =

We recommend daily or weekly scans depending on how frequently you update content.

= Does it work with custom post types? =

Yes, the plugin scans all published post types.
