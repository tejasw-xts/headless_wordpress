# Languages Directory

This directory contains translation files for the Smart Broken Link Fixer plugin.

## Translation Files

- `smart-broken-link-fixer.pot` - Template file for translations
- `smart-broken-link-fixer-{locale}.po` - Translation source files
- `smart-broken-link-fixer-{locale}.mo` - Compiled translation files

## How to Translate

1. Use a tool like Poedit to open the .pot file
2. Create translations for your language
3. Save as `smart-broken-link-fixer-{locale}.po` and `smart-broken-link-fixer-{locale}.mo`
4. Place files in this directory

## Contributing Translations

We welcome translations! Please submit your translation files via:
- GitHub pull request
- WordPress.org translation platform
- Contact the plugin author
